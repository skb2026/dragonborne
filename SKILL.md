# 暗黑破坏神掉落模拟器（5位数扩展版）

一个充满暗黑破坏神氛围的装备掉落与仓库管理系统。

### 1. 掉落规则（总范围 00000 ~ 99999）

| 稀有度   | 区间          | 概率   | 颜色 | 风格描述       |
|----------|---------------|--------|------|----------------|
| 普通     | 00000-54999   | 55%    | ⬜ 白 | 破旧、平凡     |
| 魔法     | 55000-79999   | 25%    | 🔵 蓝 | 附魔、元素     |
| 稀有     | 80000-92999   | 13%    | 🟡 黄 | 卓越、强力     |
| 传奇     | 93000-97999   | 5%     | 🟢 绿 | 史诗、传奇     |
| 太古     | 98000-99499   | 1.5%   | 🟠 橙 | 远古、禁忌     |
| **太初** | 99500-99999   | 0.5%   | 🔴 红 | 太初、灭世     |

### 2. 装备名称生成规则（暗黑破坏神风格）

使用 **前缀 + 基础装备名 + 后缀** 的组合，风格要中二、黑暗、邪恶或史诗感。

**前缀示例（按稀有度区分）：**
- 普通白装：破旧的、生锈的、普通的、劣质的、被诅咒的、腐朽的
- 魔法蓝装：寒冰的、火焰的、毒素的、闪电的、暗影的、狂暴的
- 稀有黄装：卓越的、精致的、强力的、锋利的、坚固的、浸血的
- 传奇绿装：屠夫的、毁灭者的、嗜血的、不朽的、堕落的、诅咒的
- 太古橙装：永恒的、禁忌的、混沌的、远古的、虚空的、深渊的
- 太初红装：创世者的、灭世的、唯一之、终焉的、起源的、至高的

**基础装备名示例：**
短剑、长剑、双手剑、战斧、巨斧、板甲、锁甲、皮甲、头盔、重盔、护手、战靴、戒指、护符、长弓、短弓、法杖、盾牌、项链

**后缀示例：**
之怒、之噬、之魂、杀手、屠戮者、守护、毁灭、永恒、湮灭、审判

### 3. 持久化装备仓库 + 穿戴系统

- 装备数据持久化保存在 `~/.hermes/dark_equipment.json`
- **掉落稀有及以上**时，自动询问是否存入仓库
- 支持穿戴装备（最多穿戴 6 件：武器、防具、饰品等）
- 支持查看当前已装备的套装

**支持的命令：**
- `十连`、`再来十连`、`roll10` → 进行10次掉落
- `单抽`、`roll`、`掉落` → 单次掉落
- `仓库`、`查看仓库`、`inventory` → 显示所有拥有的装备（按稀有度排序）
- `已装备`、`当前装备`、`gear` → 显示当前穿戴的装备
- `穿 [编号]` → 穿戴指定编号的装备
- `脱下 [编号]` 或 `卸下 [编号]` → 脱下指定装备
- `我的最强装备` → 显示仓库里稀有度最高的装备
- `清空仓库` → 清空所有装备（二次确认）
- `删除装备 [编号]` → 删除指定装备
