# 暗黑破坏神掉落模拟器（5位数扩展版）

一个充满暗黑破坏神氛围的装备掉落与仓库管理系统。

### 1. 掉落规则（总范围 00000 ~ 99999）

| 稀有度   | 区间          | 概率   | 颜色 | 是否唯一 | 说明 |
|----------|---------------|--------|------|----------|------|
| 普通     | 00000-54999   | 55%    | ⬜ 白 | 否       | 每次随机生成 |
| 魔法     | 55000-79999   | 25%    | 🔵 蓝 | 否       | 每次随机生成 |
| 稀有     | 80000-92999   | 13%    | 🟡 黄 | 是       | 固定唯一 |
| 传奇     | 93000-97999   | 5%     | 🟢 绿 | 是       | 固定唯一 |
| 太古     | 98000-99499   | 1.5%   | 🟠 橙 | 是       | 固定唯一 |
| 太初     | 99500-99999   | 0.5%   | 🔴 红 | 是       | 固定唯一，可献祭刷新 |

**核心规则**：
- 传奇及以上装备：**同一个数字终身对应同一件装备**，第一次生成后永久固定。
- 普通和魔法，稀有装备：每次掉落都重新随机生成（允许刷到重复名字）。
- 只有**太初**可以献祭：摧毁当前太初装备，换取一次太初区间刷新机会（生成全新的太初装备）。

### 2. 装备名称生成规则（暗黑破坏神风格）

**结构**： **[称号前缀] + [属性修饰] + [基础装备名]**

**称号前缀**（传奇及以上使用）：
屠夫的、毁灭者的、嗜血的、堕落的、永夜的、哀嚎的、禁忌的、虚空的、深渊的、古神之、创世者的、终焉的、灭世的、起源的、至高的、审判的

**属性修饰**：
嗜血的、永夜的、哀鸣的、虚空的、混沌的、浸血的、灼热的、冰封的、剧毒的、雷鸣的、锋锐的、强化过的

**基础装备名**：
短剑、长剑、双手剑、战斧、巨斧、板甲、锁甲、皮甲、头盔、重盔、护手、战靴、戒指、护符、长弓、短弓、法杖、盾牌、项链、腰带

### 3. 持久化装备仓库 + 穿戴系统

- 装备数据持久化保存在 `~/.hermes/dark_equipment.json`
- 掉落**稀有及以上**时，自动询问是否存入仓库
- 支持穿戴装备（最多6件）
- 支持查看当前已装备

**支持的命令：**
- `十连`、`再来十连`、`roll10` → 进行10次掉落
- `单抽`、`roll`、`掉落` → 单次掉落
- `仓库`、`查看仓库`、`inventory` → 显示所有装备（按稀有度从高到低排序）
- `已装备`、`当前装备`、`gear` → 显示当前穿戴的装备
- `穿 [编号]` → 穿戴指定编号的装备
- `脱下 [编号]`、`卸下 [编号]` → 脱下装备
- `我的最强装备` → 显示仓库里稀有度最高的装备
- `清空仓库` → 清空所有装备（二次确认）
- `删除装备 [编号]` → 删除指定装备
- `献祭 [编号]` → 仅限太初装备，摧毁后刷新该区间，生成全新太初装备
